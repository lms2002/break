REST API는 데이터를 주고받기 위한 표준입니다. HTTP 요청을 통해 서버에서 데이터를 조회하거나, 저장, 수정, 삭제할 수 있습니다.
GET: 데이터를 조회할 때 사용합니다.
POST: 데이터를 저장할 때 사용합니다.
PUT: 기존 데이터를 수정할 때 사용합니다.
DELETE: 데이터를 삭제할 때 사용합니다.
1.2 Spring Boot에서 API 구현
Spring Boot는 REST API를 쉽게 구현할 수 있도록 돕는 프레임워크입니다.
Controller: HTTP 요청을 처리하는 곳입니다.
Service: 비즈니스 로직을 처리하는 곳입니다.
Repository: 데이터베이스와 상호작용하는 부분입니다.


view : 읽기만 가능하고 수정이 불가능한 테이블
